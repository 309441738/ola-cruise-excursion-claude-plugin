# Cruise Excursion Options · Claude Code 插件

Explore excursion options available for cruise travelers at various ports, enhancing your travel experience.

![published with craftwave](https://img.shields.io/badge/published%20with-craftwave-6366f1)

## 安装

在 Claude Code 中执行：

```
/plugin marketplace add OWNER/ola-cruise-excursion-claude-plugin
/plugin install ola-cruise-excursion@ola-cruise-excursion-marketplace
```

> 将 `OWNER` 替换为本仓库所属的 GitHub 账号/组织名。

安装后即可在 Claude Code 里使用技能 **Cruise Excursion Options**。本仓库由
[CruiseSkillBridge](https://cruiseskillbridge.com) 按 Claude Code 插件市场规范自动生成。
